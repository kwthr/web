## Optimización de los iconos
Cuando añadas iconos a tu página, puedes considerar algunos pasos de optimización:
- Si necesitas iconos como "flecha izquierda" o "flecha derecha", puedes tomar un icono y usar el CSS para rotar el icono en la dirección que necesites. De esta manera, puedes tener un icono para varias direcciones.
- Cuando necesites cargar varios iconos, echa un vistazo a algunas aplicaciones como [IcoMoon](https://icomoon.io). Esta aplicación puede fusionar varios iconos en una sola fuente. Puede mejorar el rendimiento de la página porque hay menos peticiones. Además, la fuente es más flexible que incrustar un enlace a cada icono
- Los iconos deben estar en formato SVG. Intente evitar JPG o PNG. Esto se debe a que el tamaño de SVG se puede cambiar fácilmente sin que se vea borroso.
- Si el icono es una ruta directa pero puede cargarse de forma perezosa, debe añadir el atributo `loading="lazy"` y el atributo `decoding="async"`. Esto permite al navegador continuar renderizando la página y decodificar las imágenes de forma asíncrona.

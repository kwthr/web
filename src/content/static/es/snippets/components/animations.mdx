CSS animaciones componente utiliza Stylify CSS Keyframes comentarios opción de contenido.

Si utiliza Stylify CSS Unplugin o Bundler directamente, puede configurar CSS keyframes [dentro de la config](/es/docs/stylify/compiler#keyframes):

```javascript
stylifyVite({
	compiler: {
		keyframes: {}
	}
})
```

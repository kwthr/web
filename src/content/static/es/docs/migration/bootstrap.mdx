## Introducción
Bootstrap es un conjunto de herramientas frontend potente, extensible y repleto de características. Construye y personaliza con Sass, utiliza el sistema de cuadrícula y los componentes preconstruidos, y da vida a los proyectos con potentes plugins JavaScript.

## Configuración, personalización y variables
<MigrationCodeBlock image="bootstrap.svg">
<div slot="tool">

La personalización de Bootstrap se realiza principalmente dentro de los archivos SCSS/CSS y el tema se genera durante un proceso de construcción:

```css
// Requerido
@import "../node_modules/bootstrap/scss/functions";

// Variables por defecto
$body-bg: #000;
$body-color: #111;

// Necesario
@import "../node_modules/bootstrap/scss/variables";
```

También puedes añadir colores de tema personalizados:

```css
// Crea tu propio mapa
$custom-colors: (
	"custom-color": #900
);

// Fusionar los mapas
$theme-colors: map-merge($theme-colors, $custom-colors);
```

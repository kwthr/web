## Introducción
Foundation es un Framework para cualquier dispositivo, medio y accesibilidad. Foundation es una familia de frameworks front-end responsivos que facilitan el diseño de hermosos sitios web responsivos, aplicaciones y correos electrónicos que se ven increíbles en cualquier dispositivo.

## Componentes
<MigrationCodeBlock image="foundation.png">
<div slot="tool">

Foundation es un framework basado en componentes. Por lo tanto, hay un montón de componentes, como botones, insignias, formularios, tablas, menús, etc:

```html
<span class="badge primary">1</span>
<a role="button" class="button radius primary">1</a>
<ul class="menu">
	<li><a href="#">Uno</a></li>
	<li><a href="#">Dos</a></li>
</ul>
```

## Configuración, personalización y variables
<MigrationCodeBlock image="foundation.png">
<div slot="tool">

Los Sitios Foundation pueden ser configurados dentro del archivo SCSS. Si descargaste Foundation directamente de su sitio web, puedes configurarlo en el archivo `_settings.scss`, de lo contrario, tienes que anular las variables importadas:

```css
$foundation-palette: (
	"primary": #1779ba,
	"secondary": #767676,
	"success": #3adb76,
	"warning": #ffae00,
	"alert": #cc4b37,
);
```

## Introducción
Bulma es un framework gratuito y de código abierto que proporciona componentes frontend listos para usar que puedes combinar fácilmente para construir interfaces web responsivas.

## Componentes
<MigrationCodeBlock image="bulma.svg">
<div slot="tool">

Bulma es un framework basado en componentes. Por lo tanto, hay un montón de componentes, como botones, insignias, formularios, tablas, menús, etc:

```html
<input class="input is-primary">
<aside class="menu">
	<p class="menu-label">General</p>
```

## Configuración, personalización y variables
<MigrationCodeBlock image="bulma.svg">
<div slot="tool">

Bulma se configura a través de variables SCSS dentro de un archivo de importación y el tema se genera durante un proceso de construcción:

```css
// Configura los colores de tu marca
$purple: #8A4D76;
// ...

// Actualiza las variables globales de Bulma
$family-sans-serif: "Nunito", sans-serif;
// ...

// Actualizar algunas variables de los componentes de Bulma
$body-background-color: $beige-lighter;
// ...

// Importa sólo lo que necesites de Bulma
@import "../node_modules/bulma/sass/utilities/_all.sass";
@import "../node_modules/bulma/sass/base/_all.sass";
// ...
```

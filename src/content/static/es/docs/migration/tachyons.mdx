Esta guía está aquí para ayudarte a comparar rápidamente las características y la sintaxis del framework CSS utility-first de Tachyons con las de Stylify, y darte una idea de cómo migrar de Tachyons a Stylify.

## Introducción
Tachyons es un framework que te permite crear interfaces de carga rápida, altamente legibles y 100% responsive con la menor cantidad de CSS posible.

## Componentes
<MigrationCodeBlock image="tachyons.svg">
<div slot="tool">

Tachyons no tiene ninguna API de configuración para los componentes.
Sin embargo, tiene un conjunto preparado de componentes de copiar y pegar en su sitio web que puedes usar en tu proyecto y puedes configurarlos en el configurador mencionado en su sitio web:

```html
<div class="ph3">
	<h1 class="f6 fw6 ttu tracked">Botón básico</h1>
	<a class="f6 link dim ph3 pv2 mb2 dib white bg-black" href="#0">Texto del botón</a>
</div>

<article class="mw5 mw6-ns center pt4">
	<div class="aspect-ratio aspect-ratio--16x9 mb4">
		<div class="aspect-ratio--object cover"></div>
	</div>
</article>
```

## Configuración, personalización y variables
<MigrationCodeBlock image="uikit.svg">
<div slot="tool">

UIkit proporciona un tema que puedes modificar durante el proceso de compilación a través de variables:

```css
// 1. Tus variables personalizadas y sobreescrituras de variables.
$global-link-color: #DA7D02;

// 2. Importar variables por defecto y mixins disponibles.
@import "uikit/src/scss/variables-theme.scss";
@import "uikit/src/scss/mixins-theme.scss";

// 3. Tu mixin personalizado sobrescribe.
@mixin hook-card() { color: #000; }

// 4. Importar UIkit.
@import "uikit/src/scss/uikit-theme.scss";
```

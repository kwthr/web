Stylify puede usarse directamente en el navegador usando CDN o importarse como módulo.

<Note>El ejemplo de integración para el navegador se puede encontrar en <a href="https://github.com/stylify/integrations-examples/tree/master/browser" target="_blank" rel="noopener">repositorio de ejemplos de integración</a>.</Note>

## Instalación y configuración
```html
<script src="https://cdn.jsdelivr.net/npm/@stylify/stylify@latest/dist/stylify.min.js"></script>

<script>
// Opcional - https://stylifycss.com/en/docs/stylify/runtime
Stylify.configure({ });
</script>
```

## Instalación y Configuración - módulos
Con módulos (config es la misma que arriba):
```javascript
<script type="module">
import { Runtime } from 'https://cdn.jsdelivr.net/npm/@stylify/stylify@latest/esm/index.min.js';

// Opcional - https://stylifycss.com/en/docs/stylify/runtime
new Runtime({ });
</script>
```

Astro.build es un framework que te permite construir tu sitio web más rápido.

La integración de @stylify/astro simplifica la integración de CSS de Stylify en Astro.build.

<StackBlitzLink link="stylify-astro-example" />

## Instalación

La integración se puede instalar sólo a través de CLI como NPM o Yarn:
```
yarn add @stylify/astro
npm i @stylify/astro
```

## Uso

Añade un buildModule en el `astro.config.mjs`:

```javascript
import { defineConfig } from 'astro/config';
import stylify from '@stylify/astro';

export default defineConfig({
	integrations: [stylify()]
});
```

## Configuración

Echa un vistazo al paquete [@stylify/astro](/es/docs/astro#configuration).

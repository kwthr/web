CSS animations component uses Stylify CSS Keyframes comments content option.

If you use Stylify CSS Unplugin or Bundler directly, you can configure CSS keyframes [within the config](/en/docs/stylify/compiler#keyframes):

```javascript
stylifyVite({
	compiler: {
		keyframes: {}
	}
})
```

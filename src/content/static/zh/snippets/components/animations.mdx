CSS动画组件使用Stylify CSS Keyframes评论内容选项。

如果你直接使用Stylify CSS Unplugin或Bundler，你可以配置CSS关键帧[in the config](/zh/docs/stylify/compiler#keyframes)：


```javascript
stylifyVite({
	compiler: {
		keyframes: {}
	}
})
```

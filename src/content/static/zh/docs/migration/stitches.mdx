这篇指南是为了帮助你快速比较Stitches CSS-in-JS库和Stylify的实用优先CSS的特性和语法，并让你了解如何从Stitches迁移到Stylify。

## 简介
Stitches是一个CSS-in-JS，具有近乎零的运行时间、SSR、多变量支持，以及一流的开发者体验。

<MigrationCodeBlock image="stitches.svg">

当你想直接给某些元素设置样式时，你可以使用CSS属性：

```jsx
<Button css={{
	borderRadius: '0',
	'&:hover': {
		backgroundColor: 'black',
		color: 'white',
	},
}}>Button</Button>
```

## 全局选择器

<MigrationCodeBlock image="stitches.svg">
<div slot="tool">

全局样式可以在Stitches中定义，如下所示：

```jsx
import { globalCss } from '@stitches/react';

const globalStyles = globalCss({
	'*': { margin: 0, padding: 0 },
});

() => {
	globalStyles();
	return <div ... />
};
```

## 组件
<MigrationCodeBlock image="stitches.svg">
<div slot="tool">

Stitches允许你以这种方式定义组件：

```jsx
import { styled } from '@stitches/react';

const Button = styled('button', {
	backgroundColor: 'gainsboro',
	borderRadius: '9999px',
	fontSize: '13px',
	padding: '10px 15px',
	'&:hover': {
		backgroundColor: 'lightgray',
	},
});
```

## 配置、定制和变量

<MigrationCodeBlock image="stitches.svg">
<div slot="tool">

Stitches可以在应用程序的任何地方进行技术配置。
你可以创建一个新的Stitches实例，并在你需要的任何地方重新使用它。

全局配置的例子：
```jsx
export const { styled, css } = createStitches({
	theme: {
		colors: {
			gray500: 'hsl(206,10%,76%)',
		},
		space: {
			1: '5px',
		},
		fontSizes: {
			1: '12px',
		},
		fonts: {
			mono: 'Söhne Mono, menlo, monospace',
		},
	// ...
	}
});
```

本地配置示例：
```jsx
const { styled } = createStitches({
	theme: {
		colors: {
			violet800: 'hsl(252 62% 54.9%)',
		},
	},
});
const Button = styled('button', {
	backgroundColor: '$violet800',
});
```

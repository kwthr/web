## 简介
Foundation是一个适用于任何设备、媒介和可访问性的框架。Foundation是一个响应式的前端框架系列，它可以轻松地设计出漂亮的响应式网站、应用程序和电子邮件，在任何设备上看起来都非常棒。

## 配置、定制和变量
<MigrationCodeBlock image="foundation.png">
<div slot="tool">

Foundation网站可以在SCSS文件中进行配置。如果你直接从他们的网站下载了Foundation，你可以在`_settings.scss`文件中进行配置，否则，你必须覆盖导入的变量：

```css
$foundation-palette: (
	"primary": #1779ba,
	"secondary": #767676,
	"success": #3adb76,
	"warning": #ffae00,
	"alert": #cc4b37,
);
```

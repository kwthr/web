## 简介
Emotion是一个为使用JavaScript编写CSS样式而设计的库。它提供了强大的和可预测的样式组合，此外还提供了良好的开发者体验，如源码图、标签和测试工具。同时支持字符串和对象样式。

## 选择器和CSS工具

<MigrationCodeBlock image="emotion.png">
<div slot="tool">

在Emotion中，你可以使用CSS道具直接对元素进行样式设置，就像这样：
```jsx
<div css={css`
	color: blue;
	font-weight: bold;
	@media (min-width: 640px) {
		color: red;
	}
`}></div>
```

## 全局选择器

<MigrationCodeBlock image="emotion.png">
<div slot="tool">

有时我们需要对应用程序的某些部分进行全局风格化。为了在Emotion中做到这一点，我们需要使用Global组件：
```jsx
import { Global, css } from '@emotion/react'

render(
	<Global styles={css`
		.some-class {
			color: hotpink !important;
		}
	`}/>
)
```

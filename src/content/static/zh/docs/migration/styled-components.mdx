## 简介
Styled Components是一个可以让你在JavaScript中编写实际CSS的库。这意味着你可以使用所有你使用和喜爱的CSS功能，包括（但不限于）媒体查询、所有的伪选择器、嵌套等等。

## 全局选择器

<MigrationCodeBlock image="styled-components.png">
<div slot="tool">

当在风格化组件中定义全局样式时，你必须创建一个全局样式组件：
```jsx
import { createGlobalStyle } from 'styled-components'

const GlobalStyle = createGlobalStyle`
	body {
		color: ${props => (props.whiteColor ? 'white' : 'black')};
	}
`;

<React.Fragment>
	<GlobalStyle whiteColor />
</React.Fragment>
```

## 组件
<MigrationCodeBlock image="styled-components.png">
<div slot="tool">

在Styled Components中，组件通常是这样定义的：

```jsx
const Title = styled.div`
	color: blue
	font-weight: bold
	@media (max-width: 768px) {
		color:red
	}
`;

<Title>你好，世界! 🎉</Title>
```

<MigrationCodeBlock image="styled-components.png">
<div slot="tool">

在Styled Components CSS中，你主要使用在Javascript中定义的变量，或者一些配置文件或CSS变量。

```jsx
const color = 'blue';
const Title = styled.h1`
	color: ${color},
	font-size: var(--font-size)
`;
```

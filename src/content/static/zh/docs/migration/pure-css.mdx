这篇指南是为了帮助你快速比较Pure CSS组件框架和Stylify的实用优先CSS的特性和语法，并让你了解如何从Pure CSS迁移到Stylify。

## 简介
Pure CSS是一套小型的、响应式的CSS模块，你可以在每个网页项目中使用。

## 选择器和CSS工具

<MigrationCodeBlock image="pure-css.png">
<div slot="tool">

Pure CSS为网格等属性提供了实用工具：
```html
<div class="pure-g">
	<div class="pure-u-1-3"><p>三分之一</p></div>
	<div class="pure-u-1-3"><p>三分之一</p></div>
	<div class="pure-u-1-3"><p>三分之一</p></div>
</div>
```

## 组件
<MigrationCodeBlock image="pure-css.png">
<div slot="tool">

Pure CSS是一个框架，它提供了一些基本组件，如按钮、表单、表格、菜单等：

```html
<a class="pure-button" href="#">一个纯按钮</a>。
<table class="pure-table">
	<thead>
		<tr>
			<th>#</th>
			<th>品牌</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>1</td>

		</tr>
	</tbody>
</table>
```

## 配置、定制和变量
<MigrationCodeBlock image="pure-css.png">
<div slot="tool">

Pure CSS可以通过扩展现有的类来进行配置：

```css
/*在这个选择器中添加你的自定义样式 */
.form-custom { ... }
```

```html
<form class="form-custom pure-form"></form>
```

你也可以通过选择响应式和非响应式的Rollup来选择你要导入的组件。你可以在他们的网站上找到更多信息。

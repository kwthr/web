## 简介

Windi CSS是Tailwind的一个按需替代品，它提供了更快的加载时间，与Tailwind v2.0完全兼容，还有一些额外的很酷的功能。

## 组件
<MigrationCodeBlock image="windi.svg">
<div slot="tool">

在Windi中，组件可以像这样在一个插件中定义：

```javascript
import plugin from 'windicss/plugin'

plugin(({ addComponents }) => {
	addComponents({
		'.btn': {
			padding: '.5rem 1rem',
			borderRadius: '.25rem',
		},
	})
})
```

你也可以定义一个快捷方式：

```javascript
export default {
	shortcuts: {
		'btn': {
			'color': 'white',
			'@apply': 'py-2 px-4 font-semibold rounded-lg',
			'&:hover': {
				'@apply': 'bg-green-700',
				'color': 'black',
			},
		},
	},
}
```

## 配置、定制和变量
<MigrationCodeBlock image="windi.svg">
<div slot="tool">

在Windi中，你在配置文件中配置了一个主题。

```jsx
import { defineConfig } from 'windicss/helpers'
import colors from 'windicss/colors'
import plugin from 'windicss/plugin'

export default defineConfig({
	darkMode: 'class',
	theme: {
		extend: {
			screens: {
				'sm': '640px',
			},
			colors: {
				blue: colors.sky,
			},
			fontFamily: {
				sans: ['Graphik', 'sans-serif'],
			},
			spacing: {
				128: '32rem',
			},
			borderRadius: {
				'4xl': '2rem',
			},
		},
	},
})
```

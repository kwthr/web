## 简介

Tailwind是一个帮助你快速建立现代网站的库，而无需离开你的HTML。它是一个实用至上的CSS框架，包含了诸如flex、`pt-4`、`text-center`和`rotate-90`等类，可以直接在你的标记中组成任何设计。

## 全局选择器

<MigrationCodeBlock image="tailwind.svg">
<div slot="tool">

当你想在Tailwind中选择例如子元素时，你可以使用任意的变体来实现：

```html
<div class="
	[&>*]:underline
	[&>p]:mt-0
"></div>
```

## 组件
<MigrationCodeBlock image="tailwind.svg">
<div slot="tool">

Tailwind中的组件可以使用`@apply`规则来定义。同时，它们也可以被嵌套到CSS层中：

```css
@layer components {
	.btn-primary {
		@apply py-2 px-4 bg-blue-500 text-white;
	}
}
```

Tailwind在其网站上也提供了很多复制&粘贴的组件。

## 配置、定制和变量
<MigrationCodeBlock image="tailwind.svg">
<div slot="tool">

Tailwind可以通过扩展/覆盖现有的主题在一个配置文件中进行配置：

```javascript
module.exports = {
	content: ['./src/**/*.{html,js}'],
	theme: {
		colors: {
			'blue': '#1fb6ff',
		},
		fontFamily: {
			sans: ['Graphik', 'sans-serif'],
		},
		extend: {
			spacing: {
				'8xl': '96rem',
				'9xl': '128rem',
			},
			borderRadius: {
				'4xl': '2rem',
			}
		}
	},
}
```

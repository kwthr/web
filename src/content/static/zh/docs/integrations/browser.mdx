Stylify可以使用CDN直接在浏览器中使用，也可以作为模块导入。

<Note>浏览器的集成示例可以在<a href="https://github.com/stylify/integrations-examples/tree/master/browser" target="_blank" rel="noopener">集成示例库</a>中找到。</Note>

## 安装和配置
```html
<script src="https://cdn.jsdelivr.net/npm/@stylify/stylify@latest/dist/stylify.min.js"></script>。

<script>
// 可选的 - https://stylifycss.com/en/docs/stylify/runtime
Stylify.configure({ });
</script>
```

## 安装和配置 - 模块
有了模块（配置和上面一样）：

```javascript
<script type="module">
import { Runtime } from 'https://cdn.jsdelivr.net/npm/@stylify/stylify@latest/esm/index.min.js';

// 可选 - https://stylifycss.com/en/docs/stylify/runtime
new Runtime({ });
</script>
```

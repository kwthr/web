## 如何将 Stylify CSS 与 SolidJS 和 Vite.js 整合起来

下面的例子适用于Vite - SolidJS模板。然而，您可以使用下面的例子，并为React.js、Vue和您使用的任何其他框架配置它。

首先，使用 NPM 或 Yarn 安装 [@stylify/unplugin](/zh/docs/unplugin) 包：

```
npm i -D @stylify/unplugin
yarn add -D @stylify/unplugin
```

接下来在`vite.config.js`中添加以下配置：


```javascript
import { defineConfig } from 'vite';
import { stylifyVite } from '@stylify/unplugin';
import solidPlugin from 'vite-plugin-solid';

const stylifyPlugin = stylifyVite({
	bundles: [{ outputFile: './src/stylify.css', files: ['./src/**/*.jsx'] }],
	// 可选
	compiler: {
		// https://stylifycss.com/en/docs/stylify/compiler#variables
		variables: {},
		// https://stylifycss.com/en/docs/stylify/compiler#macros
		macros: {},
		// https://stylifycss.com/en/docs/stylify/compiler#components
		components: {},
		// ...
	},
});

export default defineConfig({
	plugins: [stylifyPlugin, solidPlugin()],
	server: {
		port: 3000,
	},
	build: {
		target: 'esnext',
	},
});
```

现在你可以将生成的`src/stylify.css`的路径添加到`src/index.js`文件中：


```javascript
import './stylify.css';
```

现在运行`yarn dev`。`src/stylify.css`文件将被生成。

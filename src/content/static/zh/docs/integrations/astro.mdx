Astro.build是一个框架，让你更快地建立你的网站。

@stylify/astro的集成简化了Stylify的CSS集成到Astro.build中。

<StackBlitzLink link="stylify-astro-example" />

## 安装

集成只能通过NPM或Yarn等CLI安装：
```
yarn add @stylify/astro
npm i @stylify/astro
```

## 用法

在`astro.config.mjs`中添加一个buildModule：


```javascript
import { defineConfig } from 'astro/config';
import stylify from '@stylify/astro';

export default defineConfig({
	integrations: [stylify()]
});
```

## 配置

查看 [@stylify/astro](/zh/docs/astro#configuration) 包。
